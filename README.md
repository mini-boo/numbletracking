# 🟢 numbletracking
방문자 수 트래킹 서비스 구축하기

<br>

# ✨ 개발 API
- url의 카운트를 증가하는 api
- 일간조회수와 누적 조회수를 응답하는 api
- 7일간의 조회수 통계 데이터를 응답하는 api (수정 진행 중)

단, 일간 조회수는 하루가 지나면 초기화 됨.

<br>

# 💻 개발 환경
Java : 11
Spring Boot : 2.7.13
Local Oracle DB : xe 11g
